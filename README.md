## ✨ Features

🚗 Smooth 3D Car Controls  
🌆 Minimal 3D City Environment  
🚦 Traffic Lights & Road Elements  
💯 Real-Time Score Counter  
🔁 Try Again Button  
🎮 Keyboard Controls  
⚡ Optimized & Lightweight

## 🛠️ Tech Stack

- 🌐 HTML5  
- 🎨 CSS3  
- 🧊 Three.js (3D Rendering Engine)

---

## 🎯 Controls

| Key | Action |
|------|--------|
| ⬅️ Left Arrow | Move Left |
| ➡️ Right Arrow | Move Right |
| 🔄 R | Restart Game |
